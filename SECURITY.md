# Security Policy

## Supported Versions

Currently, only the latest version of halo is supported for security updates. We recommend always running the most recent release to ensure you have the latest security patches.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability

If you discover a security vulnerability within halo, please do not report it publicly via GitHub issues. Instead, please email [INSERT EMAIL ADDRESS] or open a private security advisory on GitHub.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. Please provide a detailed description of the vulnerability, including steps to reproduce it.
